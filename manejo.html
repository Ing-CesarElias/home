BIENVENIDO A LA VISIÓN DE CÉSAR ELÍAS

MANEJO DE PROGRAMAS COMPUTACIONALES O SOFTWARES ESPECIALIZADOS

  • Sistemas de Información Geográficos (SIG) como por ejemplo ArcGis de ESRI, Quantum Gis.
  • HEC - HMS, modelo hidrológico para el desarrollo de análisis hidrológicos para cuencas hidrográficas rurales y urbanas.
  • HEC - RAS y HecGeo RAS, programa de aplicación para el desarrollo de análisis hidráulico para ríos y quebradas, principalmente para la obtención del nivel de aguas máximas y delimitación de zonas de protección.
  • EPANET, programa para el diseño de redes de acueductos para agua potable de urbanizaciones.
  • SWMM, programa computacional para la simulación hidráulica de las redes de aguas lluvias y canales abiertos.
  • PLUMBER V2.0 Programa para diseño de instalaciones sanitarias para agua fría y caliente para viviendas y edificaciones por el Método de Hunter.
  • DESAGUE V2.0 Programa para diseño de instalaciones sanitarias de desagüe de aguas negras para viviendas y edificaciones por el Método de Hunter
  • STADD.Pro, programa computacional para la elaboración de diseños estructurales en sistemas de concreto reforzado como estructuras metálicas varias.
  • Programas aplicativos de Windows como Word, Excel, Project, entre otros.
  • Autocad 2D y 3D y algunos aplicativos de CivilCAD para redes de agua potable.
  • Revit Básico y Revit MEP (modelación de redes hidráulicas de agua potable, negras, aguas lluvias y sistemas de combate contra incendio).

Nota: Las aplicaciones de diseños y planos hidráulicos se han enfocado en proyectos habitacionales individuales, urbanizaciones industriales y oficinas, entre otros.
